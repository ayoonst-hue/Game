# 게임 규칙: 하늘에서 떨어지는 과일 받기 (Catch Falling Fruits)

## 🎮 게임 제목
**Fruit Catcher (프루트 캐쳐)**

## 📝 게임 설명
하늘에서 쉴 새 없이 떨어지는 과일들을 바구니로 받아내는 게임입니다!
몸을 움직여 바구니를 이동시키고, 맛있는 과일들을 획득하세요. 폭탄은 피해야 합니다!

---

## 🎯 게임 목표
- 제한 시간(60초) 동안 최대한 많은 점수를 획득하세요.
- 폭탄을 피하고, 과일을 놓치지 않아야 합니다.

---

## 🕹️ 게임 규칙

### 기본 규칙
1. **시작**: "Start" 버튼을 누르면 게임이 시작됩니다.
2. **조작**: 웹캠 앞에서 몸(얼굴)을 움직여 바구니를 좌우로 이동시킵니다.
   - **왼쪽 기울임/이동**: 바구니가 왼쪽(Left) 구역으로 이동
   - **가운데(정면)**: 바구니가 중앙(Center) 구역으로 이동
   - **오른쪽 기울임/이동**: 바구니가 오른쪽(Right) 구역으로 이동
3. **득점**: 떨어지는 과일이 바구니에 닿으면 점수 획득!
4. **감점/종료**:
   - 폭탄을 받으면 점수 감점 (또는 게임 오버)
   - 과일을 바닥에 떨어뜨리면 라이프 감소

### 아이템 및 점수
| 아이템 | 설명 | 점수 |
|---|---|---|
| 🍎 **사과** | 일반 과일 | +100점 |
| 🍊 **오렌지** | 맛있는 과일 | +200점 |
| 🍇 **포도** | 희귀한 과일 | +300점 |
| 💣 **폭탄** | 피해야 할 함정! | -500점 (또는 라이프 -1) |

---

## 🎨 필요한 포즈 목록 (Teachable Machine)

이 게임을 즐기기 위해 Teachable Machine에서 다음 3가지 클래스를 학습시켜야 합니다.

| 클래스 이름 (Class Name) | 게임 동작 | 설명 |
|---|---|---|
| **LEFT** | 왼쪽 이동 | 고개를 왼쪽으로 기울이거나, 몸을 왼쪽으로 이동 |
| **CENTER** | 중앙 이동 | 정면을 바라보거나, 몸을 중앙에 위치 |
| **RIGHT** | 오른쪽 이동 | 고개를 오른쪽으로 기울이거나, 몸을 오른쪽으로 이동 |

> **Tip**: 모델 학습 시 `metadata.json`의 라벨 순서나 이름이 위와 같은지 꼭 확인해주세요!

## 📊 레벨 디자인
시간이 지날수록 게임이 더 어려워집니다.

- **Level 1 (0~20초)**: 과일이 천천히 떨어짐, 폭탄 없음
- **Level 2 (20~40초)**: 과일 속도 증가, 폭탄 등장 시작
- **Level 3 (40~60초)**: 과일 쏟아짐! 매우 빠름
